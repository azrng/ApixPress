# 清空系统数据后提示重启

## 背景
- 一键清空系统数据后，应用内运行状态虽然已同步置空，但需要给用户明确的重启选择，避免强制重启带来突兀体验。

## 变更
- 清空系统数据成功后弹出“系统数据已清空”提示，提供“稍后重启”和“立即重启”两个操作。
- 新增 `IApplicationRestartService` / `ApplicationRestartService`，立即重启时启动当前 ApixPress 主程序的新进程，并关闭当前主窗口。
- 补充清空后重启提示、稍后重启、立即重启和重启服务的自动化测试。

## 验证
- `dotnet build .\src\ApixPress.App\Apifox.App.csproj -p:OutputPath=.artifacts\verify-app\`
- `dotnet test .\tests\ApixPress.App.Tests\ApixPress.App.Tests.csproj -p:OutDir="$PWD\.artifacts\verify-tests\"`

## 备注
- 默认测试输出目录仍可能被正在运行的 `ApixPress.exe` 锁定，验证继续使用独立输出目录。
