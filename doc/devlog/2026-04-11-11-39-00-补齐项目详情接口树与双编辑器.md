# 本次目标

补齐项目详情页默认落地内容、HTTP 接口与快捷请求的双编辑器模式，以及左侧可持久化的接口树 / 用例树。

# 核心改动

- 将请求保存结构扩展为 `quick-request`、`http-interface`、`http-case` 三种类型，并支持 `folder_path`、`parent_id`
- 项目详情页默认中间区改为两个入口卡片：`新建 HTTP 接口` 和 `快捷请求`
- HTTP 接口编辑器改为环境 `BaseUrl` + 相对路径的模式，快捷请求保留自由 URL 输入
- 左侧新增默认模块接口树和快捷请求目录，HTTP 接口支持文件夹和接口下挂用例
- 增加接口层级持久化测试，验证接口、用例、文件夹路径和父子关系
