# 收紧工作区顶部工具带高度

## 本次目标
- 压缩项目工作区顶部标签工具带的高度，减少顶部区域对首屏编辑空间的占用。

## 核心改动
- 将 `ProjectWorkspaceTopActionBar` 的上下内边距从偏厚的底部留白收紧为更紧凑的工具带高度。
- 收紧工作标签 `TabStripItem` 的内边距与间距，减少标签行厚度。
- 统一压缩 `+ / ...` 按钮、环境下拉框和右侧工具按钮的高度，让整条工具带更贴合内容。
- 将 `+ / ...` 容器从顶部对齐改为垂直居中，避免局部元素继续把整行撑高。

- `src/ApixPress.App/Assets/Styles/WorkspaceSidebarStyles.axaml`
- `src/ApixPress.App/Views/Controls/ProjectWorkspaceToolbarView.axaml`

## 风险或遗留项
- 本次为工作区顶部工具带的视觉密度微调，未额外补 UI 自动化测试。
- 若后续仍觉得偏高，可继续收紧第二行的“调试 / 设计 / 文档预览”切换条。
