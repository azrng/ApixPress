# 重做项目设置导入数据入口与弹层

## 本次目标
- 将项目设置中的“导入数据”改成单一 OpenAPI / Swagger 入口。
- 点击入口后弹出导入弹层，只保留上传文件和 URL 导入两种方式。

## 核心改动
- 右侧导入数据页面移除原有平铺的上传 / URL 表单，改为单个 `OpenAPI / Swagger` 导入卡片。
- 新增项目级导入弹层，弹层内展示数据源格式说明、上传文件与 URL 导入两种模式，并复用现有 Swagger 导入命令。
- 导入成功后会自动关闭弹层，并刷新已导入数据列表。
