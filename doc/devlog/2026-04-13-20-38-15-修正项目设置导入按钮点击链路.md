# 修正项目设置导入按钮点击链路

## 本次目标
- 修正项目设置侧栏中的“导入数据”按钮点击后没有可见反馈的问题。

## 核心改动
- 将项目设置左侧“导入数据”按钮直接接到 `OpenProjectImportDialogCommand`。
- 点击后会同步切换到导入数据上下文，并直接打开导入弹层，避免出现点击后右侧无变化的空白体验。

## 修改文件
- `TASK.md`
- `src/ApixPress.App/MainWindow.axaml`

## 校验情况
- 已执行 `dotnet build Apifox.slnx -c Release -p:UseAppHost=false`
- 已执行 `dotnet test Apifox.slnx -c Release -p:UseAppHost=false --no-build`

## 风险或遗留项
- 当前点击左侧“导入数据”会直接进入导入弹层；关闭弹层后仍会停留在导入数据页面。
