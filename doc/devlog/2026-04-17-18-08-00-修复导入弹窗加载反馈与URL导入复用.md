# 本次目标

- 为 Swagger 导入弹窗补充清晰的 loading 反馈。
- 修复 URL 导入流程中部分地址因重复下载带来的不稳定问题。

# 核心改动

- 在导入弹窗中增加 busy 遮罩、进度条和导入阶段文案，导入期间锁定对话框交互。
- 在 `ProjectTabViewModel` 中补充导入 busy 文案与可编辑状态，区分 URL 校验、正式导入和结果刷新阶段。
- 在 `ApiWorkspaceService` 中为 URL 导入增加预检查结果缓存，正式导入优先复用同一次下载内容，避免同一 URL 连续请求两次。
- 补充仓储层和 ViewModel 层回归测试，覆盖 URL 复用与 loading 状态。
