# 去除通知中心 mock 并更新新版提醒文案

## 本次目标
- 将“新版提醒”开关说明改为真实启动更新提醒语义，并让右上角通知中心不再默认展示 mock 消息。

## 核心改动
- 设置中心“新版提醒”说明改为：开启后启动时自动检查更新，发现新版本会在右上角和通知中心提醒。
- 移除 `CreateNotifications()` 中的三条硬编码欢迎/演示通知，通知中心默认从空集合开始。
- 为通知中心增加真实空态：没有运行事件时显示“暂无通知”。
- `MainWindowShellPanelsViewModel` 增加 `HasNotifications` 与 `ShowNotificationEmptyState`，集合变化时同步通知。
- 更新 ViewModel 测试，确认默认无通知，并在标记已读测试中显式添加真实测试通知。
