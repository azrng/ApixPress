# 接口目录支持名称和路径搜索

## 本次目标
- 让接口管理左侧搜索框支持按接口名称和请求 Path 过滤接口目录。

## 核心改动
- 为接口目录增加搜索文本状态，输入变化后仅重建接口目录，不影响快捷请求目录。
- 过滤逻辑支持匹配接口名称、保存的接口名称和请求 URL/Path。
- 搜索时自动展开匹配结果所在的目录层级，避免命中项藏在折叠文件夹里。
