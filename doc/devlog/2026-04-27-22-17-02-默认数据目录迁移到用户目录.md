# 默认数据目录迁移到用户目录

## 本次目标
- 将默认数据库目录从程序输出目录迁移到用户本地应用数据目录，避免默认数据落到 `bin` 或发布目录下。

## 核心改动
- `AppStoragePaths.DefaultStorageDirectory` 改为用户本地应用数据目录下的 `ApixPress/data`。
- 默认设置文件路径改为用户数据目录下的 `app-shell-settings.json`。
- SQLite 默认内置连接串 `data/ApixPress.db` 改为解析到用户数据目录；若用户目录还没有数据库且旧程序目录下存在默认数据库，则只复制旧库到新位置，不删除旧文件。
- 默认设置文件迁移后仍兼容读取旧程序目录下的 `Data/app-shell-settings.json`，避免已有自定义存储目录被忽略。
- 设置中心相关文案从“本地工作目录”调整为“用户数据目录”。
- README 同步说明默认数据库目录与设置文件位置。
