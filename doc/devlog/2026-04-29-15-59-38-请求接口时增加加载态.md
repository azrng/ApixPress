# 请求接口时增加加载态

## 本次目标
- 在 HTTP 接口和快捷请求发送过程中，让返回响应区域展示明确 loading 状态。
- 请求返回、失败或取消后，恢复为响应结果或空占位状态。

## 核心改动
- 为 `ResponseSectionViewModel` 增加 `IsLoading`、`LoadingText` 以及 `BeginLoading` / `EndLoading` 状态方法。
- 在请求发送流程开始前开启响应区 loading，在流程结束时关闭 loading。
- 在 HTTP 接口工作台和快捷请求工作台的返回响应区域增加 loading 覆盖层。
- 补充响应区 loading 状态测试和请求未返回时的状态流转测试。
