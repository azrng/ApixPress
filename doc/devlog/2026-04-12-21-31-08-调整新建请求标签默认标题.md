# 2026-04-12 21:31:08 调整新建请求标签默认标题

## 本次变更
- 新建 HTTP 接口标签默认显示为 `新建 HTTP 接口`。
- 新建快捷请求标签默认显示为 `快捷请求`。
- 未命名的新建标签不再随着地址输入自动变成 `GET /xxx` 形式。

## 核心实现
- 在 `RequestWorkspaceTabViewModel` 中拆分 HTTP 接口与快捷请求的标签标题计算规则。
- 当标签是新建态且没有用户自定义名称时，优先使用固定默认标题；已有持久化记录或显式命名时仍显示真实名称。
